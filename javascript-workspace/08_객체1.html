객체

* 객체 선언 및 호출

객체는 중괄호{}를 사용해서 생성하고,
중괄호 안에 이 객체에 필요한 속성들을 정의할 수 있음
속성: 속성값의 형태로 정의함(키:밸류)
속성값으로는 모든 자료형의 값을 담을 수 있다. 

        let 변수명 = {
            속성명 : 속성값,
            속성명 : 속성값,
            ...
        };

* 객체 생성 특이케이스 (속성명 제시시 반드시 문자열 형태로 제시해야하는 케이스

* 객체에서의 반복문

객체가 가지고있는 모든 속성들에 순차적으로 접근하고자 한다면 반복문 제시
단, 단순 for loop문으로는 사용불가능하고 for in문만 사용 가능

* 객체의 메서드 속성

객체의 속성중 함수자료형인 속성을 메서드라고 부른다.
