함수(function)

* 선언적 함수

        function 함수명(매개변수, 매개변수, ....){
            해당 함수 호출시 실행할 소스코드;

            return 결과값;
        }

        - 매개변수가 없는 경우 매개변수 생략 가능
        - 반환할 결과값이 없는 경우 return 결과값 생략가능

* 익명함수

        function(매개변수, 매개변수, ...){
            소스코드;
        }

        특정 변수나 속성에 대입되는 함수 제시시 주로 사용됨
        (주로 이벤트 핸들러 작성시 사용)
